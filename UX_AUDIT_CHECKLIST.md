## Observation Template

For each step, record:
- **Time**: How long the step took
- **Confusion**: Any hesitation or wrong clicks (describe)
- **Drop-off risk**: Would a casual user quit here? (yes/no)
- **Notes**: Observations

## Funnel Steps

### Step 1: Page Load
- [ ] Page loads within 3 seconds on 4G
- [ ] No console errors on load
- [ ] Layout renders correctly (no FOUC, no layout shift)
- [ ] First meaningful paint appears quickly
- **Time to interactive**: ___s

### Step 2: Name Entry
- [ ] User immediately understands they need to enter a name
- [ ] Name input is visible without scrolling
- [ ] Submit/play button is obvious
- **Time to enter name**: ___s
- **Confusion observed**: 

### Step 3: Lobby Understanding
- [ ] User understands the game concept without explanation
- [ ] "Play" button is the most prominent action
- [ ] User is NOT overwhelmed by options
- [ ] Progressive disclosure works (minimal UI for new users)
- **Time to click Play**: ___s
- **Confusion observed**: 

### Step 4: First Round
- [ ] User understands the two concepts are displayed
- [ ] User understands they need to type a connection
- [ ] Timer is noticed but not panic-inducing
- [ ] Text input is obvious and focused
- [ ] Submit button/action is clear
- **Time to first keystroke**: ___s
- **Time to submit**: ___s
- **Confusion observed**: 

### Step 5: Score Reveal
- [ ] User understands what the score means
- [ ] Score breakdown (wit, logic, originality, clarity) is clear
- [ ] Connection explanation makes sense
- [ ] User knows how to proceed (Next Round / Share / Done)
- **Emotional reaction**: (excited/neutral/confused/frustrated)
- **Confusion observed**: 

### Step 6: Second Round Decision
- [ ] User clicks "Next Round" (not quit)
- [ ] User understands the session concept (round X of Y)
- **Did user continue?**: yes/no
- **If no, why?**: 

### Step 7: Sharing
- [ ] User notices the share option
- [ ] Share flow completes without errors
- [ ] Share text/image looks good on the target platform
- **Did user try to share?**: yes/no
