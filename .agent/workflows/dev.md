# Start Development Server

// turbo-all

1. Start the Vite dev server:
```
cmd /c "npm run dev"
```

2. The server will start on http://localhost:5173/ (or next available port)

## Hot Reloading
- Changes to React components, CSS, and assets will auto-refresh
- If you encounter issues, stop the server (Ctrl+C) and restart
